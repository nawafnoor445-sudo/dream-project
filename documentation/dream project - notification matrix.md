Dream Project — Notification Matrix (V1)
الإصدار: V1.0 (مبدئي)
التاريخ: 2026-02-08

ملاحظات
- قنوات V1: داخل التطبيق فقط.
- كل إشعار يسجل في Notifications ويظهر في Activity Feed حسب الحدث.

1) العضوية وطلبات الانضمام

حدث: إرسال طلب انضمام
- المستلم: Project Owner
- النوع: join_request
- المحتوى: "[المطور] أرسل طلب انضمام إلى [المشروع]"

حدث: قبول طلب انضمام
- المستلم: المطور
- النوع: join_accepted
- المحتوى: "تم قبول طلبك للانضمام إلى [المشروع]"

حدث: رفض طلب انضمام
- المستلم: المطور
- النوع: join_rejected
- المحتوى: "تم رفض طلبك للانضمام إلى [المشروع]"

حدث: سحب الطلب
- المستلم: Project Owner
- النوع: member_withdrew
- المحتوى: "[المطور] سحب طلب الانضمام إلى [المشروع]"

حدث: إخراج عضو
- المستلم: العضو المُخرج
- النوع: member_removed
- المحتوى: "تم إخراجك من [المشروع] (السبب: [سبب الإخراج])"

2) المهام

حدث: تعيين مهمة لمطور
- المستلم: المطور المعين
- النوع: task_assigned
- المحتوى: "تم تعيين مهمة لك في [المشروع]: [عنوان المهمة]"

حدث: نقل مهمة إلى In Review
- المستلم: Project Owner
- النوع: task_in_review
- المحتوى: "[المطور] أرسل المهمة للمراجعة: [عنوان المهمة]"

حدث: اكتمال مهمة
- المستلم: المطور المعين
- النوع: task_completed
- المحتوى: "تم اعتماد المهمة كمكتملة: [عنوان المهمة]"

3) المشروع

حدث: تغيير حالة المشروع
- المستلم: جميع أعضاء المشروع
- النوع: project_status_changed
- المحتوى: "تم تغيير حالة المشروع إلى [الحالة الجديدة]"

4) المخالفات

حدث: تحذير مخالفة
- المستلم: صاحب الحساب المعني
- النوع: violation_warning
- المحتوى: "تم تسجيل تحذير مخالفة على حسابك"

حدث: إجراء إداري (حذف/حظر)
- المستلم: صاحب الحساب المعني
- النوع: violation_action
- المحتوى: "تم تنفيذ إجراء إداري على حسابك"
